# Mac下zsh终端使用sublime打开文件及文件夹的设置

# zsh设置

  • 编辑zsh配置文件
$ vi ~/.zshrc
  • 添加如下配置内容
alias subl="open -a 'Sublime Text'"
  • 保存退出，使zsh配置生效
$ source ~/.zshrc

# 原理浅析

利用了命令"open -a 'Sublime Text' /home/Documents/xxx"
