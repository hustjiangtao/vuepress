# Git使用

# 建立临时的分支，进度同线上保持一致

git fetch git checkout -b temp origin/development

会建立线上最新代码的本地临时分支temp

  • 删除本地分支

git branch -D [branch_name]

  • 建立新分支

git branch [new_branch_name]

  • 切换分支

git checkout [another_branch_name]

  • 列出所有本地分支

git branch

  • 列出所有线上分支

git branch -a

  • 列出提交日志

git log

  • 按行列出提交日志

git log --oneline

  • 切换到历史提交版本(可查看任意的历史记录版本代码) 先找出历史提交版本的代号（可为缩短的代号） 然后切换到代号所代表的历史版本

git checkout [4ddaf91]
